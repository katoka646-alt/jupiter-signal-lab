## Project Title

Jupiter Signal Lab

## Project Description

Jupiter Signal Lab is a read-only market intelligence prototype that combines Jupiter Tokens API V2 and Price API V3 to rank Solana tokens by explainable signal quality.

Instead of only showing trending tokens, it deduplicates top-trending and top-traded tokens, pulls fresh price data, and computes an explainable signal score using liquidity, holder count, organic score, price movement, organic buy/sell pressure, trader count, and holder drift.

The project is intentionally safe for agent workflows: it does not connect a wallet, sign transactions, execute swaps, or place orders. It is a research layer that can later be extended into quote-only Swap V2 planning or Trigger order simulation with explicit human approval.
